* add icon's

Add icons in dired, emacs builtin file browser..

#+begin_src emacs-lisp

  (use-package all-the-icons-dired
    :init
    (setq all-the-icons-dired-monochrome nil)
    :config
    (add-hook 'dired-mode-hook 'all-the-icons-dired-mode))

#+end_src
